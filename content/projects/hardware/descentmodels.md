Using Autodesk 123D Make (alternate download [link](https://autodesk-123d-make.en.lo4d.com/windows)), I converted the Class 1 & Class 2 drones from the [Descent](https://www.interplay.com/) video game to physical models.

![The laser-cut Class 1 drone, assembled.](/projects/hardware-projects/laser-cut/descent-models/green_rob.jpg "Class 1 drone")
![The plans for the Class 1 drone being laser-cut.](/projects/hardware-projects/laser-cut/descent-models/green_robot.jpg "Class 1 drone plans")

![The laser-cut Class 2 drone, designed in Autodesk 123D Make.](/projects/hardware-projects/laser-cut/descent-models/bluerobot_plans.png "Class 2 drone design")
![The laser-cut Class 2 drone, assembled.](/projects/hardware-projects/laser-cut/descent-models/blue_rob.jpg "Class 2 drone")

If you'll want to download these models for yourself, be prepared for some complications!

- First, you'll need to use the [Internet Archive Wayback Machine](https://archive.org) to acquire the files.
- After that, [BRL-CAD](https://brlcad.org) is one of the few programs that can properly convert the unusual 3D model structure into *.STL* files for import into Autodesk 123D Make.
