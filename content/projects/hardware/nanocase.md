![Arduino Nano slide-on case](/projects/hardware-projects/printing/cases/Nano/nano_case.jpg "Nano Case")

This nifty little case slides on tightly, with:

- A recessed slot for the USB interface on the back
- A slot to access the interface pins (lower-right)
- A slot to see the status LEDs
- A cutout for the programming pins

The Arduino Nano was previously used to run [GRBL](https://github.com/gnea/grbl) to control a CNC engraver / mini-mill.
