## About

I wrote this program to display and manipulate [Julia](https://en.wikipedia.org/wiki/Julia_set) fractals, using direct write access to the graphics memory buffer.

This was the last Windows Mobile application I wrote, because I lost the PDA on a trip to [HPVC 2014](https://www.asme.org/events/competitions/human-powered-vehicle-challenge-(hpvc)).
