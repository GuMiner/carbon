## About

**Ski GPS Traces** focused on how to capture ski performance data and render it from downhill ski resorts.

![Ski runs overlaid on Google Earth from Whistler Blackcomb ski resort](/projects/mobile-projects/ski-gps/GoogleEarthTraces.png "Google Earth Traces")

## Process

1. Acquire GPS trace data from ski runs. Several different smartwatches, such as those used for [SimpleClarity](https://github.com/GuMiner/garmin-simple-clarity), can be used.
2. Export the data in GPX format
3. Import the data into [GPS Track Editor](http://www.gpstrackeditor.com/)
4. Delete incorrect traces, such as lift lines, and break apart each run into separate tracks.

![Ski runs in the GPS Track Editor application](/projects/mobile-projects/ski-gps/GpsTrackEditorTraces.png "GPS Track Editor")

5. Export the data as GPX format
6. Import the data into [Google Earth](https://earth.google.com/) and display as desired!
