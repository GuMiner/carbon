## About

**Specialists Calculator** was a scientific calculator with unit analysis functionality, designed for users heavily using unit analysis.

![A demo of unit analysis-based computations.](/projects/mobile-projects/specialist-calculator/sc1.png "Unit Analysis Demo")

At the time, I had a Windows Phone and wrote this software for that platform.

As that platform has been discontinued and the store shut down, this software is no longer available.

As a basic scientific calculator, this application supported:

- Trigonometric & inverse trigonometric functions
- Multiple base systems
- Logarithmic functions
- Variable storage and retrieval
- ... etc ...

![Variable storage and retrieval demo](/projects/mobile-projects/specialist-calculator/sc2.png "Variables Demo")

## Unique Aspects

This was my only phone application incorporating advertisements, and was designed to be more useful (and polished) than **Numerian** (my first Windows Phone application).

Surprisingly, incorporating advertisements was the easy part -- but due to the negligible user base, I'll never made any income from this application.
